Ohjeet / Help

  • * (asterisk) - Asteriski antaa mahdollisuuden korvata yhden tai monta kirjainta. / Matches zero or more characters.
    Example: koir* finds "koira" and "koiras"
  • % (percent) - Prosenttimerkki antaa mahdollisuuden korvata yhden kirjaimen. / Matches exactly one character.
    Example: koir% finds "koira" but not "koiras"
  • (k) - Korvaa yhden konsonantin. / Matches one consonant.
    Example: (k)oira matches "koira", "loira", etc.
  • (v) - Korvaa yhden vokaalin. / Matches one vowel.
    Example: k(v)ira matches "keira", "koira", etc.
  • l->j - Voit etsiä minimipareja, joissa on tietty kirjainvaihdos. / Search for minimum pairs with specific phoneme changes.
    Example: l->j finds pairs where 'l' changes to 'j'
